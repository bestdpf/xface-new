Getting Started

XfaceEd is written to serve as an editor for Xface Toolkit mainly. Using XfaceEd, you can specify which mesh models you want to have in your talking head (see Mesh Panel), then define the deformation schema. The result is saved under a file with the extension ".fdp" which is actually nothing but an XML file.

When you start XfaceEd, it asks for an FDP file to load or create a new one. Though you can skip this step, it is not recommended, just create an fdp file, preferrably in a new folder, so that you can also copy your static 3D mesh and texture files that you want to use with the face.

XfaceEd is distributed with a sample face Alice, and you can find and load the coresponding fdp file under "./Faces/alice/alice.fdp" in the installation folder. Since it is an XML file, you can also see the contents of the fdp file with your favorite text editor.

FDP file stores all the information that you set using XfaceEd and is used by Xface library later on. So, don't forget to save your work often, and keep a backup of your FDP file, it is "the" product of this tool!
