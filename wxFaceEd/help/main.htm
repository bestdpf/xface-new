Welcome to XfaceEd Help

Following MPEG-4 Facial Animation (FA) standard, XfaceEd provides an easy to use interface to generate MPEG-4 ready meshes from static 3D models. You can set MPEG-4 Facial Definition Points (FDP) and Facial Animation Parameter Units (FAPU), define the zone of influence of each FDP and how this influence is propagated, preview and tweak the parameters. You can also specify the key frames for the visemes, emotions and expressions.
